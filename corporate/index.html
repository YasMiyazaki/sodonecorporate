日本と世界をつなぐ架け橋

Bridge between Japan and World

Mission

日本と世界をつなぐ架け橋となるサービスを提供し、日本人が世界で、外国人が日本で活躍できる社会を実現する。

Vision

日本と外国の「違い」から生じる課題をすべて解決する。

Values

日本に来ることを夢に見て、留学、就職した結果、日本社会に慣れず帰国する外国人がたくさんいます。外国のことや外国人との接し方を知らず、ビジネスや冒険の機会を失っている日本人がたくさんいます。コミュニケーションの取り方の違いや文化の違いを知らなかったり、適切な人や情報を知らなかったことで、本来生まれるはずだったつながりを消さないために、そしてもっとつながるために、私たちは働きます。

日本の美的感覚を追求し、普及させることで、美しいと思える日本の価値観・モノ・サービスを世界に広める。

ITを使い、世界を、人と人とをより身近にします。

違いを理解し、多様性を尊重することで新しい価値を創出します。

Service

日本での生活・仕事の困りごとを解決する、外国人のためのプラットフォーム So Done -そうだん-

アナタの日本に関する悩みを相談登録すると、専門家が解決してくれます。
